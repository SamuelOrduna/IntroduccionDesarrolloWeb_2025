Tipos de campos en formularios:

Input

La etiqueta input sirve para crear campos donde los usuarios escriben o eligen datos. 
Puede ser texto, contraseña, email, número, fecha o archivo.

Label

Sirve para ponerle un nombre o descripción a un input. 
Así el usuario sabe qué debe escribir.

Output

Muestra un resultado calculado. 
Ejemplo: el total de una compra o el resultado de una suma.

Progress

Es una barra de progreso. 
Muestra cuánto falta para terminar un proceso, como una descarga.

Select

Crea una lista desplegable de opciones. 
El usuario puede elegir una o varias.

Option

Va dentro de "select". 
Sirve para poner cada opción que se puede elegir.

Textarea

Es un campo de texto largo. 
Sirve para escribir mensajes o comentarios extensos.

Propiedades de los tipos de campos en formularios:

El elemento <input> cambia según el atributo type. Si no ponemos nada, será texto simple.

Con email pedimos correos. Con password ocultamos lo que el usuario escribe. Con date o time tenemos un calendario o un reloj.

Si queremos números usamos number. Con range aparece una barra para elegir valores. Con checkbox y radio damos opciones.

file sirve para subir archivos. submit envía el formulario. reset lo borra. button es un botón normal.

Otros atributos útiles: accept limita qué archivos se pueden subir. autocomplete ayuda a rellenar datos. autocapitalize cambia mayúsculas al escribir.

7 tipos de atributo type en etiquetas input:

text

Campo básico de texto.

password

Campo de texto donde las letras se ocultan.

email

Campo que pide un correo válido.

number

Campo donde solo se aceptan números.

date

Abre un calendario para elegir una fecha.

checkbox

Casilla que se puede marcar o desmarcar.

radio

Permite elegir una opción de varias posibles.
